Урок 8. Таблицы в HTML

Основы таблиц в HTML

Таблица в HTML состоит из строк и ячеек. Основные теги:

<table> <!-- Контейнер таблицы --> <tr> <!-- Строка таблицы (table row) --> <td>Ячейка</td> <!-- Ячейка данных (table data) --> </tr> </table>

Пример простой таблицы:

Ячейка 1 Ячейка 2
Ячейка 3 Ячейка 4

Основные теги таблиц

1. <table>

Главный контейнер для таблицы. Все остальные теги таблицы должны находиться внутри него.

2. <tr> (table row)

Определяет строку таблицы. Каждая строка содержит одну или несколько ячеек (<td> или <th>).

3. <td> (table data)

Определяет стандартную ячейку таблицы, содержащую данные.

4. <th> (table header)

Определяет ячейку-заголовок. Текст внутри <th> обычно выделяется жирным и центрируется.

Пример с заголовками:

Заголовок 1 Заголовок 2
Данные 1 Данные 2

5. <caption>

Определяет заголовок таблицы. Должен идти сразу после открывающего тега <table>.

Пример с заголовком таблицы:

Моя таблица
Имя Возраст
Анна 25

Атрибуты таблиц

1. border

Определяет толщину границы таблицы в пикселях.

<table border="1">

2. width и height

Определяют ширину и высоту таблицы (в пикселях или процентах).

<table width="500" height="200">

3. cellpadding и cellspacing

cellpadding - отступ между содержимым ячейки и ее границами.
cellspacing - расстояние между ячейками.

<table cellpadding="10" cellspacing="5">

4. bgcolor

Цвет фона таблицы или ячейки.

<table bgcolor="#f0f0f0"> <tr bgcolor="lightblue"> <td bgcolor="yellow">

5. align и valign

align - горизонтальное выравнивание (left, center, right).
valign - вертикальное выравнивание (top, middle, bottom).

<td align="center" valign="middle">

Пример с атрибутами:

Таблица с атрибутами
Заголовок 1 Заголовок 2
Слева вверху Справа внизу

Объединение ячеек

1. colspan

Объединяет ячейки по горизонтали (указывает, сколько столбцов должна занимать ячейка).

2. rowspan

Объединяет ячейки по вертикали (указывает, сколько строк должна занимать ячейка).

Пример с объединением ячеек:

Объединенные столбцы
Объединенные строки Ячейка 1
Ячейка 2

Современные альтернативы

Хотя таблицы полезны для представления табличных данных, для верстки страниц лучше использовать CSS (Flexbox, Grid).
